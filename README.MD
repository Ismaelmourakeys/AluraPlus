# Alura Plus — Landing Page

![Capa do Projeto](assets/img/CapaProjeto.png)

## 📋 Sobre o Projeto

Landing page desenvolvida para o **Combo+**, plano que combina o **Alura+** e o **Alura Língua** em uma única assinatura. O projeto reproduz fielmente o layout e a identidade visual da plataforma Alura, com foco em responsividade, semântica HTML e boas práticas de CSS.

## 🚀 Tecnologias Utilizadas

- **HTML5** — estrutura semântica com `section`, `footer` e hierarquia de headings
- **CSS3** — variáveis CSS, Grid Layout, Flexbox, transições e efeitos hover
- **Google Fonts** — tipografia com a família `Inter`

## 🎨 Features

- Seções secundárias alternadas (imagem + descrição) apresentando os benefícios
- Lista de dispositivos compatíveis (TV, Computador, Celular e Tablet)
- Trabalhando com método BEM (Block Element Modifier)
- Footer com links institucionais e informações legais
- Hover animado nos botões e no logo do rodapé
- Paleta de cores com variáveis CSS reutilizáveis

## 🖥️ Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/Ismaelmourakeys/alura-plus.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd alura-plus
   ```
3. Abra o arquivo `index.html` no navegador ou utilize uma extensão como o **Live Server** no VS Code.
